Overview

This class is intended to provide NSFetchedResultsController-like functionality for use with realm-cocoa.

Parameters

objectClass

Specific RLMObject subclass used in the queries.

predicate

The predicate used for querying fetchedObjects.

sortDescriptor

The sort descriptor used for sorting the results.

sectionSortDescriptorOrNil

A sort descriptor that will be used to pre-compute the section information (must match the sectionNameKeyPath parameter).

sectionNameKeyPath

Keypath on resulting objects that returns the section name. This will be used to pre-compute the section information.

realm

The Realm instance in the main thread.

Return Value

The receiver initialized with the specified parameters.

Discussion

Initializes an instance of MRRealmResultsController.
